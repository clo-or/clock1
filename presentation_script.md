# 🎙️ 스마트제조 프로젝트 1 - 시계열 데이터 분석 대시보드 (3분 발표 대본)

**화면 구성 제안**: 
배포하신 Streamlit 웹사이트 화면을 실제로 띄워놓고 탭을 넘겨가며 시연하는 형태의 발표를 권장합니다.

---

### [도입부: 0~30초]
*화면: 메인 화면 표시 (샘플 데이터 업로드 전 빈 화면)*

**발표자**: "안녕하십니까, 스마트제조 프로젝트 1 발표를 맡은 C321050 이승아입니다. 제가 오늘 소개해 드릴 프로젝트는 공장 최적화나 수요 예측을 위해 핵심적으로 활용되는 기술인 '결측치 처리부터 시계열 데이터 분해, 그리고 sktime 머신러닝 예측까지 통합하여 지원하는 시계열 데이터 분석 대시보드'입니다. 사용자 친화적인 웹앱 환경을 제공하여 데이터분석 비전문가도 쉽게 수요 예측 모델을 돌려볼 수 있도록 고안되었습니다."

### [본론 1 - 데이터 전처리: 30초~1분 20초]
*화면: 왼쪽 사이드바에서 '샘플 데이터로 실행해보기' 버튼 클릭 후, 첫 번째 탭 [🧹 데이터 전처리] 진입*

**발표자**: "먼저, 시계열 분석의 첫 단추인 '데이터 전처리' 과정입니다. 
지금 보시는 것처럼 왼쪽 사이드바에서 CSV 파일을 직접 넣거나 샘플 버튼을 누르면 데이터가 로드됩니다. 산업 현장의 데이터는 보통 결측치가 포함된 경우가 많은데요, 보시는 것처럼 화면 좌측의 원본 데이터엔 비어있는 구간이 존재합니다.
여기서 상단의 보간법을 '선형 보간'이나 '스플라인 보간'으로 변경하시면, 파란색 선으로 결측치가 매끄럽게 채워지는 것을 확인할 수 있습니다. 뿐만 아니라 기초적인 시계열 분석을 위한 로그 변환 등 '정상성 변환'도 간편하게 시각적으로 적용해 볼 수 있습니다."

### [본론 2 - 시계열 분해: 1분 20초~2분 00초]
*화면: 두 번째 탭 [🔍 시계열 분해] 진입*

**발표자**: "전처리가 완료된 데이터를 바탕으로, 두 번째 기능인 '시계열 분해'를 수행합니다. 
데이터가 가지는 고유의 트렌드(추세)와 패턴을 파악하기 위해, 주기(Period)를 설정하면 원본 데이터를 추세(Trend), 계절성(Seasonality), 그리고 잠재 잔차(Residuals) 3가지 파트로 분리하여 보여줍니다. 이 다중 플롯 차트는 인터랙티브하게 마우스를 올려 정확한 수치 분석을 가능하게 합니다."

### [본론 3 - 머신러닝 모델 예측: 2분 00초~2분 40초]
*화면: 세 번째 탭 [🚀 모델 예측 및 평가] 진입 후, 평활법 적용 및 하단 미래 예측 스크롤링*

**발표자**: "마지막으로 이 대시보드의 핵심인 머신러닝 예측 파트입니다.
Python의 최신 시계열 라이브러리인 `sktime`을 적극 활용하여, 예측 모델 평가 파이프라인을 구축했습니다.
단순 예측이 아니라 엄밀한 검증을 위해 데이터를 Train과 Test로 알아서 8:2 분할(Temporal split)하고, Holt-Winters 평활법이나 Naive 등 선택한 모델을 통해 검증 지표를 산출해 줍니다. 
화면에 나타나는 MAE, RMSE, MAPE 지표를 바탕으로 오차율을 직관적으로 확인하며, 하단의 그래프를 통해 과거의 데이터 흐름이 왼쪽 사이드바에서 지정한 '시평(Horizon)' 파라미터만큼 미래로 어떻게 이어지는지 시각적으로 확인할 수 있습니다."

### [마무리: 2분 40초~3분]
*화면: 전체적으로 스크롤을 위아래로 한 번 부드럽게 움직이며 마무리 여운*

**발표자**: "결론적으로 본 프로젝트는, Python의 강력한 `sktime` 및 `statsmodels` 패키지가 융합된 복잡한 시계열 AI 모델링 과정을, 직관적이고 미려한 Streamlit 환경에서 원클릭으로 쉽게 사용할 수 있도록 구현하였습니다. 이를 통해 향후 제조 현장의 다양한 데이터를 분석하고 수요를 빠르게 예측하여, 경영의 효율성을 높일 수 있을 것으로 기대합니다. 
이상으로 C321050 이승아의 스마트제조 프로젝트 발표를 마치겠습니다. 감사합니다."
